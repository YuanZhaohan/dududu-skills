# 公开数据源说明

## 默认来源

使用中证指数有限公司公开的指数表现接口：

```text
https://www.csindex.com.cn/csindex-home/perf/index-perf
```

请求参数：

- `indexCode`：六位中证指数代码，例如 `000985`。
- `startDate`：开始日期，格式 `YYYYMMDD`。
- `endDate`：结束日期，格式 `YYYYMMDD`。

000985 的指数详情页：

```text
https://www.csindex.com.cn/zh-CN/indices/index-detail/000985
```

指数编制方案和事实表显示：000985 为中证全指，基日为 2004-12-31、基点为 1000，发布日期为 2011-08-02。官网接口能够返回从基日开始的回溯日线。

## 字段映射

| 官网字段 | 标准字段 | 说明 |
|---|---|---|
| `tradeDate` | `日期` | 实际交易日，不补非交易日 |
| `indexCode` | `代码` | 写成 `000985.CSI` |
| `indexNameCn` | `指数名称` | 中文简称 |
| `open/high/low/close` | `开盘价/最高价/最低价/收盘价` | 指数点位 |
| `change/changePct` | `涨跌/涨跌幅` | 官网口径 |
| `tradingVol/tradingValue` | `成交量/成交额` | 官网口径，周期计算不依赖这两列 |

周期与 LPPLS 只使用 `日期/代码/指数名称/收盘价`。其余字段用于人工核对和后续扩展。

## 校验要求

抓取后必须检查：

1. 接口返回 `code=200` 且 `success=true`。
2. 日期可解析、无重复并按升序排列。
3. 收盘价无缺失且全部为正。
4. 只包含一个指数代码和一个指数名称。
5. 写出来源接口、详情页、请求区间、实际覆盖区间、最新收盘价和抓取时间。

如果官网接口失败或返回结构变化，停止分析并保留明确错误；不要静默换源，也不要用旧缓存伪装成最新数据。用户明确允许备用源时，再单独增加来源并做末日收盘价交叉核验。
